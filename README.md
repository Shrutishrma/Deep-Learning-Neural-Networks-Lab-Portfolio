# 🧠 Deep Learning & Neural Networks: Lab Portfolio

This repository contains a comprehensive collection of labs implementations focusing on the design, optimization, and deployment of neural network architectures. The work spans from fundamental mathematical implementations of backpropagation to advanced generative and sequence-based models.

## 📂 Lab Directory - SPR_Lab

### 1. Foundations of Neural Computation
* **Lab 1: MLP & Backpropagation Architecture** (`2548554_lab-1.ipynb`)
    * **Focus:** Core implementation of Multi-Layer Perceptrons (MLP).
    * **Key Concepts:** Manual backpropagation logic, weight initialization, and activation function analysis (ReLU, Sigmoid, Tanh).
* **Lab 2: Optimization & Regularization Strategies** (`2548554_LAB_2.ipynb`)
    * **Focus:** Enhancing model convergence and generalization.
    * **Key Concepts:** Comparative analysis of SGD, Adam, and RMSprop optimizers; implementation of L1/L2 regularization and Dropout layers.

### 2. Computer Vision & Spatial Feature Extraction
* **Lab 3: Convolutional Neural Networks (CNN)** (`2548554_lab-3_DL.ipynb`)
    * **Focus:** Image classification and spatial hierarchy learning.
    * **Key Concepts:** Kernels and filters, max-pooling, flatten layers, and feature map visualization using the MNIST and CIFAR-10 datasets.

### 3. Sequence Modeling & Temporal Data
* **Lab 4.1 & 4.2: RNN, LSTM, and GRU Architectures** (`2548554_LAB_4.1.ipynb`, `2548554_lab_4.2.ipynb`)
    * **Focus:** Processing time-series and natural language sequences.
    * **Key Concepts:** Addressing the Vanishing Gradient problem, memory cells in Long Short-Term Memory (LSTM) units, and efficient sequence gating in Gated Recurrent Units (GRU).

### 4. Advanced Architectures & Generative Modeling
* **Lab 6: Unsupervised Learning & Autoencoders** (`2548554_lab-6.ipynb`)
    * **Focus:** Latent space representation and dimensionality reduction.
    * **Key Concepts:** Symmetrical Encoder-Decoder networks, bottleneck optimization, and image denoising.
* **Lab 7: Deep Learning Research Implementation** (`2548554_DL_Lab07.ipynb`)
    * **Focus:** Integration of complex DL workflows and transfer learning.
